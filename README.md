# AI Health Check — Brightcove Account Reports

Visual HTML health reports generated by the GOOSE SE Co-Pilot for Brightcove customer accounts.

## How It Works

1. Run `/jackie-pages:account-summary [account-id]` in Cowork
2. The command impersonates the account, pulls 30-day analytics, and generates a branded HTML report
3. The report is automatically committed here and published via GitHub Pages
4. A link to the live report is included in the outreach email draft

## Live Reports

Reports are published at:
`https://nveer.github.io/AI-Healthcheck/reports/[account-slug]-health-report.html`

## Setup (one-time)

```bash
git clone https://github.com/nveer/AI-Healthcheck ~/Documents/Claude/AI-Healthcheck
```

After cloning, the GOOSE account-summary command handles all future commits automatically.

## Structure

```
reports/          ← generated HTML health reports (one per customer)
.github/
  workflows/
    pages.yml     ← auto-deploys to GitHub Pages on every push
```
